## 🌟 Tentang Kami
**Core Dev Group** adalah pusat kolaborasi bagi para *programmer*, *developer*, dan antusias teknologi. Kami berfokus pada pengembangan infrastruktur komunitas modern, bot Discord interaktif, dan alat bantu produktivitas (*developer utilities*). Kami percaya bahwa kode yang baik lahir dari komunitas yang saling mendukung.

## 🤖 Proyek Utama: Core Dev Infrastructure
Proyek kebanggaan kami adalah **Core Dev Bot**, sebuah sistem automasi dan manajemen komunitas berbasis Discord yang dirancang khusus untuk para *coder*.
Fitur utama bot kami meliputi:
* **Advanced Leveling System:** Sistem XP dinamis dengan visualisasi *Rank Card* eksklusif dan promosi kasta (*Auto-Role*) secara otomatis.
* **Developer Utilities:** Pencarian instan untuk paket `NPM`, pencarian dokumentasi pemrograman `MDN Docs`, dan tracker profil `GitHub` langsung dari *command* Discord.
* **Custom Profiles:** Ekosistem database yang menyimpan *Tech Stack*, *Bio*, dan *Portfolio* para member kami.
